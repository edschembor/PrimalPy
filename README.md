PrimalPy
========

PrimalPy is a primality testing package written in Python.

Current Supported Tests
=======================

1) Trial Division
2) Fermat Test
3) Miller-Rabin Test
4) Solovay-Strassen Test

Test to be Implemented
======================

5) Pocklington Test
6) Goldwasser-Kilian Algorithm
7) ECPP Test
